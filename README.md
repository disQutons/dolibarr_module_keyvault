# Coffre-fort de clés pour [DOLIBARR ERP & CRM](https://www.dolibarr.org)

## Caractéristiques
Module permettant de stocker les identifiants et mots de passe :
- Création de clé (utilisateur, mot de passe, url)
- Mots de passe cryptés en base de données
- Vérification des accès par utilisateur

## Traductions/Translations
Les traductions peuvent être effectuées manuellement en éditant des fichiers dans les répertoires de modules sous `lang`.
Translations can be completed manually by editing files in the module directories under `langs`.
